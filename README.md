# Birko.Security.BCrypt.Tests

Unit tests for the Birko.Security.BCrypt project.

## Test Framework

- **xUnit** 2.9.3
- **FluentAssertions** 7.0.0
- **Target Framework:** .NET 10.0

## Test Classes

- **BCryptPasswordHasherTests** — Hash/verify, work factor validation, format verification, NeedsRehash, timing safety

## Running Tests

```bash
dotnet test Birko.Security.BCrypt.Tests/Birko.Security.BCrypt.Tests.csproj
```
